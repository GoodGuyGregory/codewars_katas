## Compiling  and Executing C++

[FreecodeCamp Docs](https://www.freecodecamp.org/news/how-to-compile-your-c-code-in-visual-studio-code/)

**Mac OS**

Standard Compiler

```shell
# BUILD PROGRAM:
# ==============
$ c++ -c <KataSolution.cpp>
$ c++ <KataSolution.o>

# EXECUTE PROGRAM:
# ===============
$ ./a.out
```

G++ Compiler

```shell
# BUILD PROGRAM:
# ==============
$ g++ main.cpp -o main

# EXECUTE:
# ==========
$ ./main
```
