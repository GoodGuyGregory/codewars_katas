**Running Kotlin from the CMD**

[Documentation](https://kotlinlang.org/docs/tutorials/command-line.html)

```shell
# Compiling
kotlinc hello.kt -include-runtime -d hello.jar  
```

```shell
# Running
java -jar hello.jar
```
